# ClarityForge
ClarityForge is a minimalist emotional ritual tracker built with KivyMD. It helps you design personal rituals, log emotional impact, and visualize growth across domains like clarity, sovereignty, grief, and transformation.
Whether you're healing, pivoting, or mastering self-discipline, ClosureCraft turns your emotional journey into a data-backed dashboard.
